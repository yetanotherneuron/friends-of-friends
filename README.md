# Friends Of Friends

Wrath of the Lich King addon that shares friends and ignores across alts on the same realm and faction.

## Features
- First character seeds the shared lists
- Later alts get a Sync or Keep prompt (no silent import)
- Separate friends and ignore sync
- Quiet mode and wipe / reset helpers

## Build
No build step. Copy `FriendsOfFriends/` into your WoW `Interface\AddOns\` folder.

## Commands
- `/fof` or `/friendsoffriends` — main commands
- `/fof help` — usage
- `/fof sync` — import shared friends/ignores
- `/foi` — ignore-list commands
